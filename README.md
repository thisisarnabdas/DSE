## 📦 Installation

1. Clone the repository:
   
   ```
   git clone https://github.com/thisisarnabdas/DSE.git
   ```
3. Install dependencies:
   
   ```
   pip install -r requirements.txt
   ```
5. Run the application:
   
   ```
   python main.py
   ```
